# LSPD Portal (GitHub Pages)

## Setup
1. Upload all files with the same structure.
2. Go to: Settings → Pages
3. Source: Deploy from a branch
4. Branch: main (or master) / root
5. Save

Your site will be live at your GitHub Pages URL.

## Notes
- Penal data: `assets/data/penal_codes.json`
- Traffic data: `assets/data/parking_towing_violations.json`
- If something “does nothing”, you may be opening HTML directly.
  Always test from the GitHub Pages URL.
